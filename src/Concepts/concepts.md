sudo (SuperUser DO): Ye command Pi ko batata hai ki tum Admin ki tarah kaam kar rahe ho. Bina iske, system tumhe files change nahi karne dega.

apt (Advanced Package Tool): Ye Raspberry Pi ka "Librarian" ya "Store Manager" hai jo saare softwares ko handle karta hai.

update: Ye command internet par jaakar Raspberry Pi ke official servers (Repositories) se Latest Lists download karta hai.
